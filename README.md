# Event_Countdown

## Description
Application web permettant de créer un compte à rebours vers une date et une heure précises (anniversaire, examen, voyage, etc.) et d’afficher le temps restant en temps réel.  
Ce projet est le **dixième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Manipuler les dates et heures en JavaScript
- Calculer des différences de temps
- Mettre à jour l’interface en temps réel
- Concevoir une application interactive simple

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)

---

## Fonctionnalités
- Saisie du nom de l’événement
- Sélection de la date et de l’heure
- Démarrage du compte à rebours
- Affichage du temps restant (jours, heures, minutes, secondes)
- Message lorsque le compte à rebours est terminé

---

## Design & UX
- Chiffres grands et lisibles
- Carte centrale pour l’événement
- Interface claire et moderne
- Responsive (mobile et desktop)
